## HOW TO POPULATE THE BLOCKS IN GOHIGHLEVEL

### Step-by-Step Process

#### **STEP 1: Navigate to Each Page Editor**
1. Go to `app.gohighlevel.com/v2/sites/elitecapitalrealty/pages`
2. Click **Edit** on the specific page (Properties, Agents, Blog, or Careers)
3. Wait for the page editor to load fully (may take 1-2 minutes)

#### **STEP 2: Locate the Code/Custom HTML Elements**
- Each page has **3 Code elements** (labeled as "Custom Code" or "Custom HTML/Javascript")
- They appear as placeholder boxes in the page editor canvas

#### **STEP 3: Open Code Editor**
1. Click on the first Code element on the page
2. A side panel will appear with "Custom Code" settings
3. Click **"Open Code Editor"** button
4. A full-screen code editor will open

#### **STEP 4: Copy & Paste HTML Content**
1. Open the file in `/PASTE-READY/` matching your page
2. Copy all HTML (or individual blocks from `/COMPONENTS/`)
3. Paste it into the GHL code editor
4. Click **Save** or **Apply**

#### **STEP 5: Repeat for All Blocks**
- Navigate to the next Code element if needed
- Repeat steps 3-4
- Do this for all blocks on the page

#### **STEP 6: Move to Next Page**
- Save current page
- Navigate to next page in the list
- Repeat entire process

## TROUBLESHOOTING TIPS

### Page Editor Takes Too Long to Load
- **Solution:** This is normal for GHL. Wait 2-3 minutes.
- Try refreshing the page if stuck
- Close and reopen the browser tab if needed

### Code Editor Not Opening
- **Solution:** Click directly on the Code element box first
- Wait for the side panel to appear
- Then click "Open Code Editor"

### HTML Not Displaying Correctly
- **Solution:** Make sure you copied the entire HTML block
- Check that all opening and closing tags are present
- Look for any accidental line breaks during copying

### Formatting Looks Off
- **Solution:** All CSS is inline (no external stylesheets needed)
- Images are placeholder gradients (no image URLs to update)
- Everything is mobile-responsive by default
